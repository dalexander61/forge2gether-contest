# Web App (Next.js)

This directory contains the public-facing website for the Forge2gether Essay Contest.

## Purpose
- Present the contest to students and parents
- Explain the process, timeline, and prizes
- Link to Google Forms (signup, training, submission)
- Host rules, FAQ, and privacy pages

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Phase 1 Scope
- Static pages only (no database)
- No authentication
- Clean, mobile-first design
- Environment variables for form links

## Future Enhancements
- Student dashboard
- Judge portal
- Integrated submission system
